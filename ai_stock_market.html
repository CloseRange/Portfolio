About AI Stock Market

The AI Stock Market project is an automated trading simulator developed using Node.js, OpenAI, and the Alpaca stock trading API. The program continuously pulls current news articles, processes them through OpenAI's language model, and asks it to predict how the news will influence market performance. Based on the AI's sentiment analysis and inferred market direction, the system executes simulated buy and sell orders using Alpaca. The project explores the potential of combining large language models with real-time financial data to make informed trading decisions.

Technologies

This application is built using Node.js and JavaScript, with OpenAI GPT models for natural language processing and Alpaca for market execution. It uses web scraping and API access to collect live news data, which is then summarized and interpreted by OpenAI. The model evaluates whether news is likely to affect specific stock prices and responds with buy or sell recommendations. These decisions are processed through Alpaca's paper trading API for simulated execution. The system includes logic to track portfolio performance, set trading limits, and assess historical outcomes for benchmarking prediction accuracy.

Development Process

The project began by integrating Alpaca's trading API and building a simulation environment for paper trading. News data sources were selected and parsed, then passed to OpenAI with custom prompts designed to extract actionable market insights. Iteration focused on refining the prompts to better capture financial relevance, reduce hallucinations, and handle conflicting data. The system then incorporated basic portfolio management, profit tracking, and feedback loops to evaluate the effectiveness of predictions over time. A final layer was added to filter low-confidence responses and reduce overtrading.

Usage

Users run the program in a Node.js environment with API keys for OpenAI and Alpaca. The app fetches the latest financial news, asks the AI to evaluate its potential market impact, and executes trades in a paper account accordingly. Users can observe each AI response, view buy/sell decisions, and monitor portfolio performance through logged data. It functions as both a sandbox for AI-influenced trading and a tool for exploring how language models interpret financial events.
