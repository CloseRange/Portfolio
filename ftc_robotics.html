About FTC Robotics

FTC Rorbotics is a yearly challange where you and a team work together to build a robot from scratch to compete against others in sports like events. My team, dubbed Duct Tape, also competed in this competition going all the way up to the state level and even winning a medal.

Technologies

This project mixes everything involed in robotics, from mechanical, electrical, and software engineer, to the minor things like full documentation and fundraising. This also teaches things such as welding, 3d Printing, soldering, wireing, wireless connections, mechanical fundimentals, and much more.

Java C++ FTC

Development Process

Our first prototype robot, named Daisy, was really thrown together and had no real form to it. It was a mess but ended up winning in our region. We soon upgraded and rebuilt the robot, now named Daisys Child, which was able to win on the state level. I was personally in charge of all soldering, wireing, and programming. The robot connected 2 android phones, one to the robot itself, and one to a hub that connected to an xbox controller allowing us to control it remotly

Usage

The project and all the code is over 8 years old at this point and, due to our lack of knowladge, did not keep any documentation or repositories.
