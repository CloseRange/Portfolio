About CactGuy

CactGuy is a 3D platformer developed in one week for the Kindred Games 2020 Game Jam. Built in Unreal Engine 4 using Blueprints and C++, the game features a cactus protagonist saving plant friends in a stylized environment. I focused heavily on custom shader design to achieve a distinct visual style despite limited art assets. The project required solving performance issues related to dynamic world generation and enemy spawning, balancing both technical constraints and visual goals within the tight deadline.

Technologies

CactGuy was developed in Unreal Engine 4 using a hybrid of Blueprint scripting and C++. The gameplay systems included player movement, interaction triggers, dynamic plant spawning, and area-based infection spread logic. A significant portion of the game's visual appeal came from custom shader work that enhanced lighting, outlines, and color grading to stylize the world. System performance was managed through spatial batching techniques and logic restructuring to prevent lag when spawning multiple entities across the terrain. The game was packaged and distributed via itch.io as a downloadable Windows executable.

Development Process

At the start of the jam, I prioritized visual identity and designed custom shaders to give the game a unique look that compensated for the lack of detailed 3D assets. Core mechanics such as jumping, object collection, and area-of-effect infection logic were built with rapid prototyping in mind. The biggest challenge was designing a dynamic spawning system for both friendlies and enemies that could scale with minimal performance impact. After several profiling passes and system rewrites, I settled on a layered spawning solution that balanced performance with flexibility. The short timeline required daily iteration and fast communication between myself and my teammate.

Usage

CactGuy can be downloaded from its itch.io page, where users can extract the game folder and run DadJam.exe. While the page suggests a donation, it is not required to download. Once launched, players control CactGuy using keyboard or controller input, navigating a series of small environments to find and rescue fellow plant characters while avoiding hazards. The game highlights the creative potential of limited-scope development under time constraints.
