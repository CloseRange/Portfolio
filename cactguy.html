About CactGuy

CactGuy is a short 3D platformer game where you play as a cactus who just wants to save his plant friends.

Technologies

This project was made using Unreal Engine 4 with a mix of Blueprints and C++

Unreal Engine C++

Development Process

CactGuy was made for the Kindred Games 2020 Game jam in the span of 1 week

Usage

To download, visit the itch.io page and hit download. It may ask for a donation however this is not required. You will need to extract the given folder and run the file 'DadJam.exe'
