About Portfolio Maker

Portfolio Maker is an application developed by me simply for creating the portfolio you are viewing now. It allows you to add projects, images, and descriptions with ease streamling the process of updating and maintaining a portfolio.

Technologies

This project is made using Java and ImGui for the UI. Of course I also had to use HTML, JavaScript, CSS, SASS, and JQuery for the website itself.

JavaScript Java Portfolio Maker HTML Sass JQuery

Development Process

In college I knew I needed to make a portfolio but I wanted to make it as easy as possible, so I ended up overcomplicating it and writing an entire application that would generate the website for me so that now updating the website or making changes takes only a few minutes

Usage

To use, download the project from the github page and run the .exe file provided. Alternativly you can use the jar file, or compile the Java code yourself.
