About AI Card Generator

This project is more of a proof of concept. This uses AI to generate on the fly, including a picture, name, and description of the card. This can further be used for games or other applications. As this was mostly a test, it currently only generates cute anime girls.

Technologies

This project relies heavily on OpenAI but also uses GML for the front end because it's the quickest way for me to throw together a good looking interface.

ChatGPT GML

Development Process

My idea was to try and make a game out of this project. Perhaps a game where you can create cards for a card game instead of just getting predefined ones. Your card that you own would be unique to you, nobody else in the world would have it, but you'd also be able to trade cards with others.

Usage

As this was just a tech demo, no code or usage is provided. If you want to try and recreate this for yourself, it's quite simple. Just generate a detailed description of the character you want, send it to Chat GPT. Use that same descriotion to also have it generate you a name, and a description. Plug it all in and you have the outline for a card.
