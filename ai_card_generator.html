About AI Card Generator

The AI Card Generator is a prototype tool designed to generate collectible-style cards themed around anime-style characters. Developed in GameMaker Language (GML), the program integrates with OpenAI to take user-defined prompts and generate character descriptions, which are then rendered into card designs. This tool explores the intersection of natural language processing and game content creation, offering a proof of concept for procedural narrative and visual generation in card-based game systems.

Technologies

The project is built in GameMaker Studio using GML for core logic and layout design. It leverages the OpenAI API to interpret prompt input and return detailed character descriptions and stats. These outputs are formatted into a visual card layout featuring themed artwork, generated or selected based on the AI's interpretation. While primarily a tech demo, the system demonstrates dynamic UI generation, text parsing, and AI-driven character creation in a lightweight game engine.

Development Process

The project began as an experimental system to test how AI could drive content generation in games. The core was built in GML, where I created a framework for card layout, user prompt input, and formatting logic. I connected the OpenAI API to parse user prompts and return a themed character description. Image assets were mapped to AI responses based on key traits or tags, enabling dynamic card visuals. The development focused on quick iteration and creative flexibility, highlighting how AI can be embedded into visual game tools.

Usage

Users enter a prompt describing a desired character, and the system sends that to OpenAI. The AI returns a customized character concept, which is then displayed on a themed card with text and visuals. Users can generate multiple cards and explore how different prompts affect output. The tool serves as a sandbox for AI-assisted creativity, targeting use cases like card game prototyping or narrative character inspiration.
