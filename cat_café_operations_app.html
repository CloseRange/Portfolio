About Cat Café Operations App

This mobile application is designed to streamline daily operations for a cat café business. It provides tailored tools for both managers and employees to handle everything from employee scheduling and safe counts to inventory control and cat roster management. The app integrates with Square to facilitate secure, trackable clock-ins and clock-outs, helping ensure accurate payroll reporting. Aimed at simplifying and digitizing operations, this solution serves as the central operational hub for the café staff.

Technologies

The app is built using Flutter for cross-platform mobile development, ensuring smooth performance on both Android and iOS. Supabase powers the backend, offering PostgreSQL database functionality, real-time syncing, file storage for cat images, and role-based authentication for secure access. To manage employee time tracking, the app integrates with the Square API, allowing employees to clock in and out using QR code verification, with data syncing directly to Square's payroll system. A custom CRImage class was developed to handle image cropping, uploading, and linking efficiently, especially for cat profiles. The app’s modular architecture uses stateful widgets and page-based routing to maintain clean code organization and easy feature expansion.

Development Process

The development started with identifying key operational pain points for café managers and staff, followed by wireframing user roles and workflows. I structured the app into core modules employee, inventory, financials, cat management and used Flutter's widget-based architecture for clean modularity. Supabase was selected for its fast setup, real-time capabilities, and scalable auth/storage. Square's developer sandbox was used for safe testing of payroll and clock-in logic. I also implemented a custom CRImage class for optimized cat image handling. Testing and validation were conducted iteratively with simulated store workflows.

Usage

The app is designed for internal use by cat café staff, divided into manager and employee roles. Managers use the app to add and manage employee accounts, set store hours, perform daily safe counts, update inventory logs, and maintain a digital roster of the café's resident cats. They can upload and edit cat images, include adoption notes, and track individual care needs. Employees, on the other hand, can securely clock in and out using the built-in Square integration with QR code verification, view their upcoming shifts, update personal profile info, and report low inventory or supply issues. While not yet publicly released, the app is fully functional in testing and intended for full deployment within the café's daily operations.
