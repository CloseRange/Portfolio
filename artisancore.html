About ArtisanCore

ArtisanCore is a 2D game engine with the intent of turning game development into a breeze for me and for others. The application is still under heavy development

Technologies

This app uses Java as it's main language with ImGui for the UI. I am also using Rhino to allow for JavaScript as a scripting language. While glfw is the backend for rendering images, the entire game renderer and all the back end systems were developed by me.

Development Process

I have attempted to make game engines from scratch before but never to this extent. This project started as just a way for me to quickly develope games without going through the hassle of redeveloping a game engine every time or being forced to use one of the pre-existing engines

Usage

As of right now ArtisanCore is heavily under development so there is no way to use the application however the plan is to allow users to download ArtisanCore straight from Steam and start development right away
